# Playbook: Content Creation

## When to Use
Writing blog posts, LinkedIn posts, email newsletters, video scripts, social media content, or any external communications.

## Steps

### 1. Clarify the Brief
- What format? (LinkedIn post, blog, email, script)
- What audience? (B2C members, B2B employers, investors, general)
- What's the goal? (awareness, conversion, engagement, education)
- Any specific angle or data point to include?

### 2. Load Context
- Read `reference/brand-voice.md` for tone and language rules
- Check `brain/master.md` Section 7 for content strategy
- If referencing product features, read Dean's `VYVEBrain/brain/master.md` Section 2

### 3. Write
- Always in VYVE brand voice: warm, human, evidence-based
- UK English (programme, colour, wellbeing, organisation)
- "Members" not users/customers
- Never "Corporate Wellness"
- Include a clear CTA where appropriate
- For LinkedIn: hook in first line, value in body, CTA at end

### 4. Review Checklist
- [ ] Brand voice consistent?
- [ ] UK English throughout?
- [ ] No promises about features that aren't live (check Dean's brain if unsure)?
- [ ] CIC explained if mentioned (first use only)?
- [ ] HAVEN not referenced as a live feature?
- [ ] Contact email is team@vyvehealth.co.uk?

### 5. Log
- Save to Command Centre > Content section (if using CC)
- Or commit to repo for reference
